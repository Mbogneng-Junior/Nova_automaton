# Prompt : Génération de cover

## Format

1:1 pour thumbnail YouTube / TikTok
9:16 pour les Shorts

## Exemple prompt

```
Afrofuturistic French artist, neon Paris night,
luxury aesthetic, anime realism, dark blue atmosphere,
cinematic, 4K, square format 1:1
```

## Consignes

- Palette : bleu nuit (#0A0E1A) + or (#C9A84C)
- Style : afrofuturisme, néons, silhouettes, grain cinématographique
- Pas de texte dans l'image
