# Workflow Template

Template pour créer un nouveau workflow dans Automaton.

## Objectif

Décrire ici le cas d'usage métier.

## Entrées

- Source de données : Google Sheet, Notion, API, webhook...
- Paramètres utilisateur

## Sorties

- Type de contenu produit
- Plateformes de distribution

## Chaîne de traitement

1. Déclencheur
2. Génération de concept
3. Génération de contenu
4. Stockage
5. Rendu/Formatage
6. Publication
7. Analytics

## Prompts

Voir `prompts/`.

## Workers

Aucun worker spécifique par défaut. Utiliser l'API générique et le FFmpeg worker si besoin.
