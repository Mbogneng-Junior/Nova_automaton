# Prompt de génération de concept

## Rôle

Tu es un expert en [domaine].

## Tâche

Générer un concept complet pour [type de contenu] à partir des paramètres fournis.

## Format de sortie

Retourner un JSON structuré avec :
- `title`
- `prompt`
- `seo_title`
- `seo_description`
- `tags`
- `thumbnail_prompt`

## Exemple

```json
{
  "title": "...",
  "prompt": "...",
  "seo_title": "...",
  "seo_description": "...",
  "tags": ["..."],
  "thumbnail_prompt": "..."
}
```
